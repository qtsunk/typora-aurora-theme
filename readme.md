# Aurora Theme for Typora

一款专为 **技术写作** 与 **开发者** 设计的 Typora 主题。

Aurora 汲取了极光的灵感，以冷静、理性的蓝灰色调为主轴，构建了一个既专业又舒适的写作环境。它摒弃了繁杂的装饰，专注于内容本身的呈现，旨在为技术文档、代码笔记和深度文章提供最佳的阅读体验。

## ✨ 设计理念

- **专业与克制**：采用严谨的排版布局，确保标题、段落与代码块之间的呼吸感，让复杂的文档结构依然清晰有序。
- **代码优先**：深知代码是技术文档的核心，因此特别选用了优质的等宽字体，并精心调配了语法高亮配色，让每一行代码都清晰易读。
- **沉浸式体验**：无论是白天的清爽明亮，还是夜晚的深沉静谧，Aurora 都能提供一致且流畅的视觉流，助您进入心流状态。

## 🌗 双模式支持

- **Aurora (Day)**：明亮而不刺眼，适合日常办公与文档分享。
- **Aurora Night**：深邃护眼，专为夜间编码与沉浸式写作打造。

## 📦 安装

1. 下载主题文件 [`aurora.css`](aurora.css) 和 [`aurora-night.css`](aurora-night.css)。
2. 打开 Typora，进入 **偏好设置** > **外观** > **打开主题文件夹**。
3. 将下载的 `.css` 文件复制到该文件夹中。
4. 重启 Typora，在主题菜单中选择 **Aurora** 即可。

## ⚙️ 自定义

主题代码结构清晰，基于 CSS 变量构建。如果您有独特的配色偏好，可以直接在文件顶部的 `:root` 区域进行微调，打造属于您的专属风格。
