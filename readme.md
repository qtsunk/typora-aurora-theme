**Aurora Typora Markdown theme** 是我个人优化并使用的 Typora 主题。灵感源于宁静的极光，这款主题通过柔和的蓝色调点缀、精心设计的排版层次和舒适的阅读体验。

## 特色

- **优雅的视觉层次**：精心调整的标题层级与间距，引导读者自然流动
- **柔美色彩系统**：以宁静蓝为主色调，搭配中性灰，减轻视觉疲劳
- **性能优化**：轻量级 CSS，不影响 Typora 运行性能

## 安装

1. 下载主题文件 [`aurora.css`](aurora.css)
2. 打开 Typora，进入 **偏好设置** > **外观** > **打开主题文件夹**
3. 将下载的 `aurora.css` 文件复制到打开的主题文件夹中
4. 重启 Typora，在主题菜单中选择 **Aurora** 即可应用

## 夜间模式 (Night)

- 已新增主题文件 `aurora-night.css`，基于 Aurora 的暗色配色，适合夜间阅读。
- 启用：重启 Typora，进入 **偏好设置 > 外观**，在主题列表中选择 `aurora-night`。
- 自定义：主题使用 CSS 变量，你可以编辑 `:root` 中的变量来自定义配色。

如果你使用便携版或自定义路径，请把文件复制到对应的 `themes` 文件夹下。

## 自定义选项

Aurora 主题采用 CSS 变量设计，您可以通过修改 `:root` 部分轻松调整整体风格：

```css
:root {
    --primary-color: #4a6fa5;    /* 主色调，影响链接、引用等元素 */
    --border-color: #e9ecef;     /* 边框颜色 */
    --table-header-bg: #f0f5fb;  /* 表头背景色 */
    --quote-border-color: rgba(74, 111, 165, 0.6); /* 引用区块左侧竖线颜色 */
    /* 更多变量请查看主题文件 */
}
```
